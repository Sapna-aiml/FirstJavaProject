# ID Card Generator System

## Description : This is a Java Swing based ID Card Generator System.
It allows ussers to enter student details and generate a ID card with a photo.

## Features
- Enter Student Name
- Enter ID Number
- Enter Department
- Enter Email
- Mobile Number
- Course
- UPload student Photo
- Generate ID card
- reset form
- input Validation

## Technologies used
- Java
- Java Swing
- IntelliJ IDEA

## How to Run
1. Open the project in IntelliJ or VS code.
2. Open Main.java.
3. Run the project.
4. Fill all details.
5. Click **ID Card Generate**.

## Output : Student details and Photo are displayed on the genrated ID Card.
